## 独立开发与部署

本仓库已包含 Cloudflare Worker 配置，可单独开发和部署。

```bash
pnpm install
pnpm dev
```

部署：

```bash
pnpm deploy
```

Worker 配置文件：`workers/gbti-worker/wrangler.toml`
